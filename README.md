# Brew Tap for Cloud.Gov

This is repo holds the homebrew formulae for utilities from [cloud.gov](https://www.cloud.gov)

## Installation
```
brew install rbogle/cloudgov/<formula>
```

or

```
brew tap rbogle/cloudgov
brew install <formula>
````
